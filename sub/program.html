사전예약프로그램

  • 청춘의 설렘 새로운 교방

    교방 문화의 시(詩) / 서(書) / 화(畵)를 배워보고,
    작품을 만들어보는 체험

    stat_0날짜 : 5. 3.(토) ~ 5. 6.(화)

    stat_0장소 : 진주성 촉석루

    * 예약인원 : 회차당 최대 7명

    * 1인당 최대 2명까지 신청 가능

    * 1일 1개 프로그램만 신청 가능

    1. 프로그램 선택

      • 시(詩)
      • 서(書)
      • 화(畵)

    2. 날짜 선택

      • 5월 3일 토요일
      • 5월 4일 일요일
      • 5월 5일 월요일
      • 5월 6일 화요일

    3. 시간 선택

      • 1회차(14:00 ~ 15:00) 7명/7명
      • 2회차(16:00 ~ 17:00) 7명/7명
      • 1회차(11:00 ~ 12:00) 7명/7명
      • 2회차(13:30 ~ 14:30) 5명/7명
      • 3회차(15:00 ~ 16:00) 7명/7명
      • 4회차(17:00 ~ 18:00) 7명/7명

  • 예약마감

    수상레저 – 카약 체험

    가족, 친구와 함께 하는 즐거운 카약 체험

    stat_0날짜 : 5. 3.(토) ~ 5. 6.(화)

    stat_0장소 : 성지동 둔치 앞 남강

    * 진주성 촉석문 아래 계단으로 이동

    * 예약인원 : 회차당 20팀

    * 1대당 최대 2명까지 탑승 가능

    * 성인 기준 대당 1~2명,
    9세 미만 어린이 포함시에만 3명까지 탑승 가능

  • 진주성 해설사 투어

    어린이 해설사와 문화관광해설사가
    들려 주는 진주성 이야기

    stat_0날짜 : 5. 3.(토) ~ 5. 6.(화)

    stat_0장소 : 진주성 공북문 앞에서 출발

    * 5. 3.(토) 14:00 / 5. 5.(월) 14:00 는 일부 코스를 어린이 해설사가 해설합니다

    * 예약인원 : 회차당 최대 20명
